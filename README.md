# Astro 5.16.7

## Instalar Astro 

``` bash
  npm create astro@latest
  pnpm create astro@latest
```
  - Nombre de la carpeta. 
  - Elegir un template --> 'Use minimal (empty) template.
  - Instalar dependencias.
  - Elegir si iniciar un repositorio en github.
  - Levantar el servidor "pnpm run dev / pnpm dev".
  - Agregar la extencion en vsc *Astro language support extencion*.

### Primeros
  **Nueva pagina**
  - para agregar una nueva pagina debera de crearse dentro de la carpeta pages
      - src/pages/nameFile.astro
  - Podemos agregar mas carpetas dentro de la carpeta pages
    - src/pages/posts/nameFile.md
    - Donde la salida del url sera localhost:4321/posts/nameFile.
